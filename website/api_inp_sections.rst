API for the Input File Sections
===============================

Import from
`swmm_api.input_file.helpers.sections <https://gitlab.com/markuspichler/swmm_api/-/blob/master/swmm_api/input_file/inp_section/labels.py>`_
all section headers, so you don't have to write every string for your own and save some typo error handling.

.. code-block:: python

    from swmm_api.input_file.helpers import sections

Each section has a specific type, which can be seen under:
`swmm_api.input_file.helpers.sections <https://gitlab.com/markuspichler/swmm_api/-/blob/master/swmm_api/input_file/inp_section/types.py>`_
